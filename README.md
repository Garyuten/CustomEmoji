# CustomEmoji
slackなどに使えるカスタム絵文字です。

主に[絵文字ジェネレーター](https://emoji-gen.ninja/)で作成しています。

## カスタム絵文字をまとめて登録する方法

1. Chrome拡張 [Slack Custom Emoji Manager \- Chrome ウェブストア](https://chrome.google.com/webstore/detail/slack-custom-emoji-manage/cgipifjpcbhdppbjjphmgkmmgbeaggpc/related) をインストール
2. ブラウザでSlackのEmoji管理ページへアクセス https://xxxxxx.slack.com/customize/emoji
3. 追加したい絵文字をドラック&ドロップ
